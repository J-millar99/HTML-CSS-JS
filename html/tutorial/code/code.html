키보드 입력을 위한 HTML <kbd>

<kbd>는 문서에서 일부 텍스트를 키보드 입력으로 정의합니다.
Save the document by pressing Ctrl + S

프로그램 출력을 위한 HTML <samp>

<samp>는 문서에서 컴퓨터 프로그램의 샘플 출력으로 일부 텍스트를 정의합니다.

Message from my computer:

File not found.
Press F1 to continue

HTML <code> 컴퓨터 코드

<code>는 문서에서 일부 텍스트를 컴퓨터 코드로 정의합니다.

x = 5; y = 6; z = x + y;

변수에 대한 HTML <var>

<var>는 문서에서 일부 텍스트를 변수로 정의합니다.

The area of a triangle is: 1/2 x b x h, where b is the base, and h is the vertical height.
