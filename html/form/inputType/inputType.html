HTML에서 사용할 수 있는 입력 유형은 다양합니다.

예시) <input type="text"> 속성의 기본값은 text입니다.

입력 유형 텍스트

<input type="text"> 단일 줄 텍스트 입력 필드를 정의합니다.



입력 유형 비밀번호

<input type="password"> 비밀번호 필드를 정의합니다.



입력 유형 제출

<input type="submit"> 폼 핸들러에 폼 데이터를 제출하기 위한 버튼을 정의합니다.





입력 유형 재설정

<input type="reset"> 모든 양식 값을 기본값으로 재설정하는 재설정 버튼을 정의합니다.





입력 유형 라디오

<input type="reset"> 모든 양식 값을 기본값으로 재설정하는 재설정 버튼을 정의합니다.

Choose your favorite Web language:



입력 유형 체크박스

<input type="checkbox"> 체크박스를 정의합니다.


입력 유형 버튼

<input type="button"> 버튼을 정의합니다:

입력 유형 색상

<input type="color"> 색상을 포함해야 하는 입력 필드에 사용됩니다.

입력 유형 날짜

<input type="date"> 날짜를 포함해야 하는 입력 필드에 사용됩니다.

min또한 및 max속성을 사용하여 날짜에 제한을 추가할 수 있습니다.



입력 유형 Datetime-local

<input type="datetime-local">는 날짜 및 시간 입력 필드를 지정하며, 시간대는 지정하지 않습니다.

입력 유형 이메일

<input type="email">이메일 주소를 포함해야 하는 입력 필드에 사용됩니다.

입력 유형 이미지

<input type="image"> 이미지를 제출 버튼으로 정의합니다.

입력 유형 파일

<input type="file">는 파일 선택 필드와 파일 업로드를 위한 "찾아보기" 버튼을 정의합니다 .

입력 유형 숨김

<input type="hidden"> 사용자에게 보이지 않는 숨겨진 입력 필드를 정의합니다. 사용자에게 표시되지 않지만, 개발자 도구나 소스보기에서 볼 수 있으니, 보안의 형태로 사용하지 말 것!


입력 유형 월

사용자는 <input type="month"> 월과 연도를 선택할 수 있습니다.

입력 유형 숫자

<input type="number"> 숫자 입력 필드를 정의합니다. 허용되는 숫자에 대한 제한을 설정할 수도 있습니다.

입력 유형 범위

<input type="range"> 허용되는 숫자에 대한 제한을 설정할 수 있습니다.(기본값은 0 ~ 100)

입력 유형 검색

<input type="search"> 검색 필드에 사용됩니다(검색 필드는 일반 텍스트 필드처럼 작동합니다).

입력 유형 전화

<input type="tel"> 전화번호를 포함해야 하는 입력 필드에 사용됩니다.

입력 유형 시간

<input type="time"> 시간(시간대 없음)을 선택할 수 있습니다.

입력 유형 URL

<input type="url"> URL 주소를 포함해야 하는 입력 필드에 사용됩니다.

입력 유형 주

<input type="week"> 주와 연도를 선택할 수 있습니다.
