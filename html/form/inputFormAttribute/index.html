폼 속성

input의 form 속성은 input요소가 속한 양식을 지정합니다.

input의 form속성 값은 해당 속성이 속한 form의 id요소와 동일해야 합니다. <form action="https://www.w3schools.com/action_page.php" id="form1"> <label for="fname">First name:</label> <input type="text" id="fname" name="fname"><br><br> <input type="submit" value="Submit"> </form> HTML form 외부에 있는 입력 필드이지만 여전히 form의 일부이다. <label for="lname">Last name:</label> <input type="text" id="lname" name="lname" form="form1">


formaction 속성

input의 formaction 속성은 양식이 제출될 때 입력을 처리할 파일의 URL을 지정합니다.

<form action="/action_page.php"> <label for="fname">First name:</label> <input type="text" id="fname" name="fname"><br><br> <label for="lname">Last name:</label> <input type="text" id="lname" name="lname"><br><br> <input type="submit" value="Submit"> <input type="submit" formaction="https://www.w3schools.com/action_page.php" value="Submit as Admin"> </form>




formenctype 속성

input의 formenctype 속성은 제출 시 양식 데이터를 어떻게 인코딩해야 하는지 지정합니다(method="post"인 양식에만 해당).

<form action="/action_page_binary.asp" method="post"> <label for="fname">First name:</label> <input type="text" id="fname" name="fname"><br><br> <input type="submit" value="Submit"> <input type="submit" formenctype="multipart/form-data" value="Submit as Multipart/form-data"> </form>


formenctype 속성

input의 formmethod 속성은 form-data를 action URL로 전송하기 위한 HTTP 메서드를 정의합니다.
참고: 이 속성은 form 요소의 method 속성을 재정의합니다

<form action="https://www.w3schools.com/action_page.php" method="get"> <label for="fname">First name:</label> <input type="text" id="fname" name="fname"><br><br> <label for="lname">Last name:</label> <input type="text" id="lname" name="lname"><br><br> <input type="submit" value="Submit using GET"> <input type="submit" formmethod="post" value="Submit using POST"> </form>




formtarget 속성

input의 formtarget속성은 양식을 제출한 후 수신된 응답을 표시할 위치를 나타내는 이름이나 키워드를 지정합니다.
참고: 이 속성은 form 요소의 대상 속성을 재정의합니다.

<form action="https://www.w3schools.com/action_page.php"> <label for="fname">First name:</label> <input type="text" id="fname" name="fname"><br><br> <label for="lname">Last name:</label> <input type="text" id="lname" name="lname"><br><br> <input type="submit" value="Submit"> <input type="submit" formtarget="_blank" value="Submit to a new window/tab"> </form>




formnovalidate 속성

input의 formnovalidate속성은 input 요소가 제출될 때 유효성 검사를 거치지 않아야 함을 지정합니다.
참고: 이 속성은 form 요소의 novalidate 속성보다 우선합니다.

<form action="https://www.w3schools.com/action_page.php"> <label for="email">Enter your email:</label> <input type="email" id="email" name="email"><br><br> <input type="submit" value="Submit"> <input type="submit" formnovalidate="formnovalidate" value="Submit without validation"> </form>


novalidate 속성

novalidate 속성은 form의 속성 입니다.
참고: novalidate가 존재하는 경우, 모든 양식 데이터가 제출될 때 검증되지 않아야 함을 지정합니다.

<form action="https://www.w3schools.com/action_page.php" novalidate> <label for="email">Enter your email:</label> <input type="email" id="email" name="email"><br><br> <input type="submit" value="Submit"> </form>
