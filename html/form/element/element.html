input 요소

가장 많이 사용되며, type 속성에 따라 input 요소를 여러 가지 방법으로 표시할 수 있습니다.

select 요소

select는 드롭다운 목록을 정의합니다.

        size는 보여지는 선택지의 개수이며,
        multiple은 다중 선택을 허용하는 것입니다.
        다중 선택은 컨트롤이나 커맨드 키를 사용해야 합니다.

textarea 요소

textarea는 다중 줄 입력 필드를 정의합니다.

css를 사용하여 텍스트 영역의 크기를 정의할 수도 있습니다.

button 요소

button은 클릭 가능한 버튼을 정의합니다.

fieldset, legend 요소

fieldset는 양식에서 관련 데이터를 그룹화하는 데 사용됩니다.
legend는 fieldset에 대한 캡션을 정의합니다.

Personalia:




datalist 요소

datalist는 미리 정의된 옵션 목록을 지정합니다.
사용자는 데이터를 입력할 때 미리 정의된 옵션의 드롭다운 목록을 보게 됩니다.
input 요소의 list 속성은 datalist요소의 id 속성을 참조해야 합니다.

output 요소

output은 계산의 결과를 나타냅니다.
